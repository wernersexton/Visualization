Illustraing JavaScript Code in HTML

We create a simple HTML document that displays two images. The viewer can change the image on the left using a pull-down menu. She can hide and display the image on the right by toggling the checkbox.

We create the HTML with a text editor. We'll display the two images in a table, but we could use CSS with float: left and float: right. The user-interface elements are within a form element, but this is not essential as we use JavaScript event handlers on each element. The two form elements are SELECT and an INPUT. We specify JavaScript a command(s) for the onchange attribute on each of these. The select element has several OPTION elements giving the items in the pull down menu. The value attribute of each OPTION gives the URL for the image, either a regular URL or a local file name. We use these as the value for the corresponding IMG element's src attribute.

The JavaScript code is stored in a separate file named Eg.js

The CSS file for this HTML document is in eg.css.
